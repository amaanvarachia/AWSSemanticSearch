A modern, enterprise-level Data Engineering + AI Retrieval project leveraging Amazon Bedrock, Amazon OpenSearch Serverless, Amazon Web Services Lambda, and Vector Embeddings.
**AI-Powered Semantic Search Platform — AWS Serverless Architecture**

- Designed and implemented a **fully serverless, AI-driven semantic search platform** leveraging modern AWS technologies including Lambda, Bedrock, S3, and OpenSearch Serverless.
- Built an end-to-end data ingestion pipeline to process raw customer datasets and generate **LLM-grade vector embeddings** using **Amazon Bedrock Titan**.
- Architected and deployed a **vector database** using **AWS OpenSearch Serverless (AOSS)** with HNSW-based KNN indexing for high-performance semantic similarity search.
- Developed a real-time **semantic search API** using AWS Lambda, including embedding generation, vector retrieval, and relevance scoring.

Implemented secure, scalable infrastructure with **IAM least-privilege**,**SigV4-signed API calls**, and custom Lambda layers for optimized dependency management.
